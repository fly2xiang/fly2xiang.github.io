总结

这里使用 Java 编写了第一个 MapReduce 程序。MapReduce 程序也可以使用其他语言进行编写：

  • Hadoop streaming 可以使用任何的可执行文件作为 Mapper 和 Reducer，例如 shell，C/C++ 程序，Python程序。
  • Hadoop Pipes 使用 SWIG 让 C/C++ 可以编写 MapReduce 程序。
